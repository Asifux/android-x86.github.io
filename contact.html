Google group (mailing list)

To participate this project, send your test reports, patches or suggestions to the discussion group:

  • https://groups.google.com/group/android-x86

Before sending technical problems to the mailing list, make sure you have read all related documents carefully, and did some study yourself (e.g., ask google). If you want to ask a troubleshooting issue, please provide enough information, including what your target device is, what branches you tried, and debug messages like output of dmesg and logcat.

Do not ask if we support your device or not. We usually don't have a chance to test it (unless you'd like to donate one). Just try it yourself. If it doesn't work well, let us know.

Issues tracker

Send bug reports or features request to the issues tracker
  • https://github.com/android-x86/android-x86.github.io/issues

IRC channel

To communicate with us real-time, joinn the irx channel #android-x86 at irc.freenode.net
You need an iirc client, like irssi, a Linux text mode client. Here is a simple instruction:
$ mkdir android-x86
$ cd android-x86
$ repo init -u git://git.osdn.net/gitroot/android-x86/manifest -b $branch
$ repo sync --no-tags --no-clone-bundle
$ irssi -c irc.freenode.net
